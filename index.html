Common HTML Tags

HTML is a markup language used to create web pages, and is the standard for the content of web pages. Below is a list of some common HTML tags you should know other than the basic tags like <h1>, <p>, <img>. Knowing these tags is very important for web development as they are building blocks of big web pages. Of course, you can find more tags on MDN Web Docs.

<Details & Summary> Tag

Used to create collapsible sections

Demo:


            <details>
                <summary>Click Here</summary>
                    <h1>Main Heading</h1>
                    <p>Details</p>
            </details>

CSS Techniques

CSS is easy when you look at it at the beginning, however, it can be very challenging when you go into a complex project. therefore, it is important to have a good understanding of CSS Techniques not just the basic properties but also how to use them, below is a list of some CSS Techniques you should know very well.

Flexbox Layout

Using flexbox to create a flexible layout

Demo:


                <div style="display: flex;">
                    <div style="flex: 1; height: 100px; background-color: red;"></div>
                    <div style="flex: 1; height: 100px; background-color: green;"></div>
                    <div style="flex: 1; height: 100px; background-color: blue;"></div>
                </div>

Grid Layout

Animating elements with CSS
